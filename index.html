SplatWalk

About SplatWalk

SplatWalk is a browser-based utility for turning 3D Gaussian splat scenes into game-ready assets: exportable .glb ground meshes and Recast-compatible navmesh binaries. .ply is the primary supported input format; .spz import is available but not yet fully supported.

Use FAST NAV to extract a walkable 2.5D floor field and bake click-to-move navigation in one click, or the advanced voxel collision workflow for manual Recast tuning. Ground meshes can be reconstructed as a RANSAC single plane or a voxelized ground surface. Processing runs in Rust WASM with Babylon.js visualization.

3D Workbench

Drag and drop a .ply splat on the canvas (.spz experimental), use the file chooser, or select an example file to begin.
