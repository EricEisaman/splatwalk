# @splatwalk/core-pro

![SplatWalk](https://raw.githubusercontent.com/EricEisaman/splatwalk/main/public/splatwalk.png)

> **Reserved placeholder.** The commercial Pro tier of SplatWalk, **releasing in
> conjunction with SplatWalk v1.0.0.** There is no functional code in this version
> yet - it reserves the package name.

The free, MIT-licensed core is
**[`@splatwalk/core`](https://www.npmjs.com/package/@splatwalk/core)** - everyone
gets a free start. `@splatwalk/core-pro` adds advanced, opt-in capabilities for
teams that need the extra splat juice, under a commercial license. The free core
never depends on Pro.

## Planned for v1.0.0

- **Streaming splat-storage adapter generator** - ready-to-deploy streaming
  adapters for popular backend object stores.
- **GPU-assisted shN palette clustering** - accelerate the slowest stage of SOG
  export on very large scenes.
