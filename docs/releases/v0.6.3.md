# SplatWalk v0.6.3

## Highlights

- **Storage Adapter Region and prune**: edit camera-select AABB offsets (meters:
  left, right, forward, behind, below, above) and click **Apply select region from
  camera** to pin the yellow box from the live fly view.
- **Oval interior** preset fly pose updated to
  `pos=[-329.834, 4.212, 148.185]`, `eulerDeg=[28.5, 79.2, 0.0]`.

## Notes

- Builds on the host `cameraSelect` API from **0.6.2**.
- WASM **`api_version` stays `2`**.

## Upgrade

```bash
npm install @splatwalk/core@0.6.3
```
