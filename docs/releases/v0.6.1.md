# SplatWalk v0.6.1

## Highlights

- **Camera-based select region** helper: `regionBoundsFromCameraPose` (defaults 10 m L/R, 15 m forward, 5 m behind, 5 m below / 15 m above) → world AABB for Viewer / WASM `region_min`·`region_max`.
- **Oval interior** Storage Adapter example auto-arms that region after the initial fly pose.
- Other CDN examples keep prior region behavior (manual toggle / suggestRegion / budget pin).

## Notes

- WASM **`api_version` stays `2`**. No new WASM capability; the helper is host-side.
- Babylon FreeCamera yaw 0 looks along **+Z** (left-handed).

## Upgrade

```bash
npm install @splatwalk/core@0.6.1
```
