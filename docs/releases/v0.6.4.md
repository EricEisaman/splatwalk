# SplatWalk v0.6.4

## Highlights

- **Upload nav artifacts** on every demo (Storage Adapter, 3D Workbench, Vuetify Fast Nav, R3F): one `.zip` or multi-select of contract filenames (`nav_session.json` + volume trio **or** `recast.navmesh.bin`).
- **Download nav artifacts** on Workbench / Vuetify / R3F (minimal Recast pack) and Storage (full volume pack after voxel nav).
- **3D Workbench**: Upload button to the right of Choose local file; Region/prune camera-select AABB offsets + Apply from camera (same idea as Storage 0.6.3).

## Notes

- WASM **`api_version` stays `2`**.
- R3F upload requires `recast.navmesh.bin` (volume walk restore is Babylon/Storage-first).

## Upgrade

```bash
npm install @splatwalk/core@0.6.4
```
