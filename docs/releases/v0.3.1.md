# SplatWalk v0.3.1

Patch release.

## Fixed

- **Manual/non-OIDC `npm publish` no longer fails on provenance.** `provenance: true`
  is no longer baked into the published package's `publishConfig` (it aborted any
  publish outside a supported OIDC CI provider). Provenance is now enabled only in
  CI via `NPM_CONFIG_PROVENANCE=true` in the release workflow, so tagged CI
  releases keep their signed provenance while a local bootstrap publish works too.

## Added

- **Reserved [`@splatwalk/core-pro`](https://www.npmjs.com/package/@splatwalk/core-pro)** -
  a placeholder for the commercial Pro tier, releasing in conjunction with
  SplatWalk v1.0.0. The free [`@splatwalk/core`](https://www.npmjs.com/package/@splatwalk/core)
  remains MIT, free forever, and never depends on Pro.

## Install

```bash
npm install @splatwalk/core
```
